# Starter Prompts

These are copy-paste conversation starters for each step of the Hacky Hours framework. Use them to kick off a Claude Code session without staring at a blank cursor wondering what to say.

Each prompt is designed to:
- Give Claude the context it needs to be helpful
- Tell Claude what you want to accomplish in the session
- Get Claude asking you questions rather than guessing

---

## How to Use These

1. Open your project folder in the terminal
2. Run `claude` to start a session
3. Copy the relevant prompt below, paste it into Claude, and press Enter
4. Answer Claude's questions — that's the work

You don't have to use these verbatim. Modify them to fit your situation. The goal is to give Claude a clear starting point.
