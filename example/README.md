# Example Project: NeighborBoard

This folder contains a fully worked example of the Hacky Hours framework applied to a fictional project called **NeighborBoard** — a simple community event and announcement board for neighborhoods.

It is not a real app. It exists to show you what filled-in documents look like at each level, so you have a concrete model to work from when filling in your own.

## How to Use This

1. Open any file here to see what a completed version looks like
2. Compare it to the blank template in the corresponding folder (`01-ideate/`, `02-design/`, etc.)
3. Use it as a reference when filling in your own project's documents

When you're working with Claude and you're not sure how much detail to include, you can say: "Here's an example of a completed document from the NeighborBoard example project. I want something similar for my project."
